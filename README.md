🚀 Vendor Management System

A RESTful backend application developed using Spring Boot for managing vendor records with full CRUD operations, input validation, and database integration.

🛠 Tech Stack

Java 17 (LTS)

Spring Boot

Spring Data JPA

Hibernate

H2 Database

Maven

IntelliJ IDEA

Postman

📌 Key Features

RESTful API design

Layered architecture (Controller → Service → Repository)

Create, Read, Update, Delete operations

Input validation using @Valid

Structured response handling

H2 in-memory database integration

▶ Running the Application

Open project in IntelliJ IDEA

Ensure JDK 17 is configured

Run the main class

Server starts at:

http://localhost:8081
🔗 Base Endpoint
http://localhost:8081/vendor
